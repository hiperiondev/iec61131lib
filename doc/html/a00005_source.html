41/*
42 * Summary:
43 *
44 * Function Parameter Type Parameters Description
45 *
46 * ADD_DT_TIME Adds an interval to a date and time value.
47 * ADD_TOD_TIME Adds an interval to a time of day value.
48 * ADD_TIME Adds two interval values.
49 * CONCAT_DATE_TOD Adds a time of day value to a date.
50 * CONCAT_DATE Constructs a DATE value from: Year, Month and Date values.
51 * CONCAT_DT Constructs a DATE_AND_TIME value from: Year, Month, Date, Hour, Minute, Second and Millisecond values.
52 * CONCAT_TOD Constructs a TIME_OF_DAY value from: Hour, Minute, Second and Millisecond values.
53 * DAY_OF_WEEK Returns the day of week of a DATE value.
54 * DIVTIME Divides an interval by a numeric value.
55 * MULTIME Multiplies an interval by a numeric value.
56 * SPLIT_DATE Extracts the fields from a DATE value.
57 * SPLIT_DT Extracts the fields from a DATE_AND_TIME value.
58 * SPLIT_TOD Extracts the fields from a TIME_OF_DAY value.
59 * SUB_DATE_DATE Subtracts two date values.
60 * SUB_DT_DT Subtracts two date and time values.
61 * SUB_DT_TIME Subtracts an interval from a date and time value.
62 * SUB_TOD_TOD Subtracts two time of day values.
63 * SUB_TOD_TIME Subtracts an interval from a time of day value.
64 * SUB_TIME Subtracts two interval values.
65 *
66 */
