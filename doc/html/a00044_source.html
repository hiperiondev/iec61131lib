41/*
42 * Summary:
43 *
44 * Function Parameter Type Parameters Description
45 * TO ANY 1 Simple conversion, changing from type * to type **
46 * TRUNC (note 1) 1 Real to integer conversion, changing from real type * to integer type **
47 * BCD_TO (note 2) 1 Change from Binary Coded Decimal (BCD) type * to integer type **
48 * TO_BCD (note 3) 1 Change from integer type * to Binary Coded Decimal (BCD) type **
49 *
50 * note 1: The from (*) type can be REAL or LREAL. The to (**) type can be SINT, USINT, INT, UINT, DINT, UDINT, LINT, or ULINT
51 * note 2: allowed
52 * BYTE_BCD_TO_SINT: Convert two BCD digits encoded in a BYTE to a small integer (SINT).
53 * WORD_BCD_TO_INT: Convert four BCD digits encoded in a WORD to an integer (INT).
54 * DWORD_BCD_TO_DINT: Convert eight BCD digits encoded in a DWORD to a double integer (DINT).
55 * LWORD_BCD_TO_LINT: Convert sixteen BCD digits encoded in a LWORD to a long integer (LINT).
56 * note 3: allowed
57 * SINT_TO_BCD_BYTE: Convert a small integer (SINT) to two BCD digits encoded in a BYTE.
58 * INT_TO_BCD_WORD: Convert an integer (INT) to four BCD digits encoded in a WORD.
59 * DINT_TO_BCD_DWORD: Convert a double integer (DINT) to eight BCD digits encoded in a DWORD.
60 * LINT_TO_BCD_LWORD: Convert a long integer (LINT) to sixteen BCD digits encoded in a LWORD.
61 */
