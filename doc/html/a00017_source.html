45 *
46 * Function Parameter Type Parameters Description
47 * ABS ANY_MAGNITUDE 1 Return the absolute value
48 * SQRT ANY_REAL 1 Return the square root
49 * LN ANY_REAL 1 Return the natural logarithm
50 * LOG ANY_REAL 1 Return logarithm base 10
51 * EXP ANY_REAL 1 Return natural exponent
52 * SIN ANY_REAL 1 Return the sine of the input. The input is in radians
53 * COS ANY_REAL 1 Return the cosine of the input. The input is in radians
54 * TAN ANY_REAL 1 Return the tangent of the input. The input is in radians
55 * ASIN ANY_REAL 1 Return the principle arc sine of the input. The output is in radians
56 * ACOS ANY_REAL 1 Return the principle cosine of the input. The output is in radians
57 * ATAN ANY_REAL 1 Return the principle tangent of the input. The output is in radians
58 * ATAN2 ANY_REAL, ANY_REAL 2 Return the principle tangent of the input. The output is in radians.
